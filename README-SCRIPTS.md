# Scripts de Utilidad

## start-server.ps1
Inicia el servidor de desarrollo con verificaciones previas.

**Uso:**
```powershell
.\start-server.ps1
```

## check-connection.ps1
Verifica la configuración de la base de datos y la resolución DNS.

**Uso:**
```powershell
.\check-connection.ps1
```

## Solución de Problemas

Si los comandos fallan con timeout:
1. Ejecuta los scripts manualmente desde PowerShell
2. O ejecuta los comandos directamente:
   ```powershell
   cd RodMarInventory
   npm run dev
   ```
